# Telegram Bot

This is a simple Telegram bot deployed to GitHub Actions to ensure it stays responsive.

## Features
-   Responds to `/start` and `/help` commands.
-   Echoes back any message sent to it.
-   Runs 24/7 using GitHub Actions.

## Setup
1.  Clone the repository.
2.  Install dependencies: `pip install -r requirements.txt`.
3.  Run the bot: `python bot.py`.
